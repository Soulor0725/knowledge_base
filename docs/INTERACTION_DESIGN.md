## 四、交互规范

### 4.1 页面切换

| 操作 | 动画 | 时长 |
|------|------|------|
| 菜单切换 | 淡入淡出 | 0.2s |
| 弹窗打开 | 缩放+淡入 | 0.2s |
| 弹窗关闭 | 缩放+淡出 | 0.15s |
| Toast显示 | 从右滑入 | 0.3s |
| Toast消失 | 淡出 | 0.2s |

### 4.2 表单交互

| 场景 | 交互 |
|------|------|
| 输入聚焦 | 边框变色 #667eea |
| 输入错误 | 边框变红 + 提示文字 |
| 提交成功 | Toast + 页面跳转 |
| 提交失败 | Toast + 保留表单 |

### 4.3 列表交互

| 场景 | 交互 |
|------|------|
| 鼠标悬停 | 背景变化 + 阴影 |
| 点击项目 | 进入详情 |
| 收藏切换 | 图标变化 + Toast |
| 删除确认 | 弹窗确认 |

### 4.4 搜索交互

| 场景 | 交互 |
|------|------|
| 输入搜索 | 300ms防抖 |
| 搜索中 | Loading状态 |
| 有结果 | 列表更新 |
| 无结果 | 空状态提示 |

## 七、动画规范

### 7.1 过渡效果

```css
/* 按钮悬停 */
.btn:hover {
    transform: translateY(-2px);
    box-shadow: 0 4px 12px rgba(102, 126, 234, 0.4);
    transition: all 0.2s ease;
}

/* 卡片悬停 */
.card:hover {
    transform: translateY(-4px);
    box-shadow: 0 4px 16px rgba(0,0,0,0.1);
    transition: all 0.2s ease;
}

/* 页面切换 */
.page {
    animation: fadeIn 0.2s ease;
}

@keyframes fadeIn {
    from { opacity: 0; }
    to { opacity: 1; }
}
```

### 7.2 加载动画

```css
/* 骨架屏 */
.skeleton {
    background: linear-gradient(90deg, #f0f0f0 25%, #e0e0e0 50%, #f0f0f0 75%);
    background-size: 200% 100%;
    animation: shimmer 1.5s infinite;
}

@keyframes shimmer {
    0% { background-position: 200% 0; }
    100% { background-position: -200% 0; }
}
```

## 八、导入蓝湖/墨刀指南

### 8.1 蓝湖导入

1. 将HTML原型截图或设计稿上传到蓝湖
2. 标注各组件的尺寸、间距、颜色
3. 添加交互说明和跳转链接
4. 生成分享链接给开发团队

### 8.2 墨刀导入

1. 创建新项目，选择"Web端"
2. 按照页面流程创建各页面
3. 使用组件库搭建界面
4. 添加交互事件和动画
5. 发布原型链接

### 8.3 需要标注的信息

| 信息类型 | 内容 |
|----------|------|
| 尺寸 | 宽度、高度、间距 |
| 颜色 | 背景色、文字色、边框色 |
| 字体 | 字号、字重、行高 |
| 交互 | 点击、悬停、跳转 |
| 状态 | 默认、悬停、禁用、错误 |
