## v2.2.0 - 2026-06-28

### 🐛 重要修复

#### 🔧 数据库连接泄漏修复
- 修复 `close_db` 未注册 `@app.teardown_appcontext` 装饰器的问题

- 影响场景：加班/消费模块的删除、批量删除接口在并发或异常路径下偶发失败

### ✨ 功能优化

#### 📚 工作日报文案适配
- 文章模块在「工作日报」分类下，所有用户可见文案中的「文章」自动切换为「日报」

  - 列表编辑/删除按钮 tooltip
  - 删除确认、保存成功、加载失败提示
  - 上一篇/下一篇导航到头提示
  - 批量删除确认与成功提示
  - 空列表状态文案
- 顶部统计栏「文章」改为「日报」
- 搜索框占位符「搜索文章...」改为「搜索日报...」
- 修复弹窗标题短暂闪烁问题

#### 🎨 UI 全面美化

**布局与间距**
- 压缩侧栏模块间距，减少滚动
- 用户头像放大（32/36px → 44px）
- 顶部统计栏移至「工作日报」分类内部显示

**文章/日报列表**
- 表头升级：13px/600/亮色 + 深色背景
- 标题列升级：14px/亮色/500 字重
- 操作按钮放大 + hover 蓝/红语义色

**全系统统一**
- 数据行 hover、操作按钮、分页、空状态、状态标签、表格圆角全系统统一
- 配色：金额黄色→青色，未发货黄色→蓝色，加班时长绿色→蓝色

## v2.1.0 - 2026-06-27

### 🐛 重要修复

#### 🔧 导出功能编码问题修复
1. **消费记录导出**
   - 修复Excel打开乱码问题
   - 改用GBK编码（Windows Excel原生支持）
   - 新增UTF-8 BOM标记确保兼容性

2. **猕猴桃销售订单导出**
   - 从前端生成改为后端API处理
   - 新增 `/api/kiwi-sales/export` 后端API
   - 使用GBK编码，完美支持中文显示

#### 🍋 猕猴桃销售模块优化
- 优化导出流程，支持全选和多选导出
- 无选择时导出当前月份数据
- 支持分类筛选后的导出

#### 📋 消费记录模块功能
- 新增消费记录导出功能
- 支持多选、全选、无选择导出当前月
- 完整的消费管理功能

### 🔧 技术改进
- 新增 `codecs` 导入处理编码
- 使用 `BytesIO` + `StringIO` 正确处理CSV流
- 统一Content-Type为 `text/csv; charset=gbk`

## v2.0.0 - 2026-04-26

### 🎉 重大更新

#### 🌟 项目更名
- 项目名称从"个人知识库"更名为"智慧管理中心"
- 更准确地反映了系统的综合管理特性

### ✨ 新增功能

#### 🍋 猕猴桃销售管理模块
1. **订单管理**
   - 支持创建、编辑、删除销售订单
   - 客户名称、运单号码管理
   - 规格选择（5斤装/10斤装）

2. **数据字段**
   - 数量管理（整数类型）
   - 支付金额（保留2位小数）
   - 发货状态（已发货/未发货）

3. **实用功能**
   - 导出订单到Excel（CSV格式）
   - 批量删除订单
   - 订单选择导出后自动清除选择并隐藏删除按钮

4. **报表统计**
   - 按客户分组统计
   - 按规格汇总统计
   - 总数量和总金额统计
   - 分页显示，每页10个客户
   - 实时数据同步，订单变更后自动更新报表

#### 📚 知识库模块优化
- 保留所有原有知识库功能
- Markdown编辑器、代码高亮
- 用户认证、分类管理、收藏功能
- 搜索、浏览统计、响应式设计

### 🔧 技术优化

#### 前端优化
1. **布局调整**
   - 优化销售订单列表布局
   - 防止运单号码折行
   - 报表列宽优化，数量列固定80px防止被挤压
   - 金额列自动调整宽度，支持大金额显示

2. **UI/UX改进**
   - 统一表头字体大小为15px
   - 优化报表分页控件
   - 改善按钮点击反馈

#### 后端优化
1. **API增强**
   - 新增销售订单管理API
   - 新增销售报表统计API
   - 支持分页查询

2. **数据库**
   - 自动数据库迁移
   - 新增字段自动添加

### 🐛 问题修复

- 修复订单编辑后数据不同步问题
- 修复删除后删除按钮不自动隐藏问题
- 修复多选功能被意外移除的问题
- 修复备注列不显示内容的问题
- 修复数据库路径错误问题

### 📦 项目配置

- 更新启动信息为"智慧管理中心"
- 更新README.md完整文档
- 项目结构优化整理

### 🚀 使用说明

1. **启动服务**
   ```bash
   python app.py
   # 或
   start.bat
   ```

2. **访问地址**
   - 本地：http://localhost:5001
   - 内网：http://[你的IP]:5001

3. **功能导航**
   - 左侧菜单切换知识库/销售模块
   - 销售模块下有"销售订单"和"销售报表"两个子菜单
